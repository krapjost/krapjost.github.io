주차장 관리

개발 기간

2020.08.17~2020.08.28

개요

자동화 주차장 시스템을 구상하고 구현해보았다.
입차부터 출차까지의 과정에 관리자가 신경써야 할 부분을 최소화 하고, 또한 사용자의 입장에서도 이용이 최대한 직관적으로 느껴지게 만들고 싶었다.

frontEnd

서버 사이드 렌더링으로 ejs템플릿을 사용하여 동적 웹을 만들었다. 구상 단계부터 핸드폰으로 모든 과정을 해결할 수 있게끔 만들고 싶었기에 렌더링 시 DB에서 데이터를 받아와 주차 가능한 공간을 실시간으로 띄워주도록 만들었다. 입차하면 입차 시간이 DB에 저장되며 해당 정보를 받아와 주차 시간을 실시간으로 계산하여 보여준다. 그 시간 정보를 프론트 단에서 금액으로 환산하여 출차 시 해당 금액을 결제하도록 만들었다.

backEnd

node.js, express 프레임워크를 사용하여 구축하였다. 데이터베이스는 mysql을 사용하였고 디자인 패턴은 MVC로 설계했다. starUML을 사용하여 데이터베이스 구조를 설계했다. 하지만 개발 도중에 설계의 결함이 발견되어 처음부터 다시 만들기도 했다.

잊지마! 언더워터

Android needs to setup SHA certificate on project settings page on firebase. to get SHA for debug, type

keytool -list -v \ -alias androiddebugkey -keystore ~/.android/debug.keystore

default password is 'android'

'Firebase는 두 가지 유형의 인증서 지문을 지원합니다. SHA-1: 앱에 대한 API 키와 OAuth 2 클라이언트를 만드는 데 사용됩니다. SHA-256: Firebase 동적 링크를 구성하는 데 사용됩니다.'

copy paste these keys and download google-services.json again.

you probably should do something more on product mode! don't forget!
